# 📚 Prototip — Образовательная платформа

Платформа для учителей и учеников с функциями:
- 📖 Теория с видео, аудио и файлами
- 📝 Домашние задания (текст, пропуски, квизы)
- ✅ Автопроверка заданий
- 👨‍🏫 Ручная проверка учителем с оценками и комментариями
- 📊 История попыток и детальные результаты

## 🚀 Запуск локально

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_НИК/prototip.git
cd prototip

# 2. Создайте виртуальное окружение
python -m venv .venv
.venv\Scripts\activate  # Windows
# source .venv/bin/activate  # macOS/Linux

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите приложение
python app.py

# 5. Откройте в браузере
http://127.0.0.1:5000
